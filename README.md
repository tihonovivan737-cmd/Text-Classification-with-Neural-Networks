Проект реализует классификацию текстовых новостей на 4 категории (мир, спорт, бизнес, наука/технологии) с использованием различных архитектур нейронных сетей:

LSTM (Long Short-Term Memory)
GRU (Gated Recurrent Unit)
MLP (Multi-Layer Perceptron)
Transformer

Результаты
По результатам исследования:

GRU показала наилучшее соотношение точности (80.55%) и скорости обучения (4 минуты)
Transformer показал хорошую точность (80.1%), но требует больше ресурсов
LSTM достигла точности 78.05% за 7 минут обучения
MLP склонна к переобучению (72% на тестовой выборке)
